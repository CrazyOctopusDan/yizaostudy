# 一级造价师教材目录树任务设计

## 背景

当前系统已经能按知识点生成每日学习任务，但用户反馈“知识点”仍然不够像官方教材目录，学习者不能直观看到整本教材哪些章节已完成、哪些未完成。新版本需要把任务主线改成教材目录树，让学习者每天既能看到今日推荐，也能像游戏地图一样看到整科教材进度。

## 目标

新版本以公开教材目录和公开备考计划为依据，把四科任务组织为：

- 科目。
- 章。
- 节。
- 目/知识点。

系统需要在“学习计划”页展示树形任务列表。每个叶子节点支持 `未开始`、`已学`、`模糊`、`掌握` 四种状态；章和节自动汇总完成率。今日任务从目录树中选择下一个未完成或模糊节点，完成后同步更新树形进度、角色经验、科目进度和复盘队列。

## 来源边界

目录以公开可查资料整理，不复制教材正文。2026 年公共科目及土建/安装专业教材公开资料显示沿用 2025 版教材；2025 版教材由中国计划出版社、中国城市出版社等发行。目录结构参考 233 网校公开教材目录、2025 版考试大纲和公开备考计划资料。

## 计划规则

系统保留最后一个月冲刺期：

- 2026-05-25 至 2026-09-16：完成已勾选科目的教材目录树第一轮学习。
- 2026-09-17 至 2026-10-16：只安排真题、套卷、错题、模糊点回炉和案例采分训练，不再推进新教材节点。

排期按剩余可学习天数和未完成叶子节点动态计算。越靠近考试，每日推荐任务数和练习强度越高；前期以养成规律为主，每个科目每天最多推进一个新教材节点。

## 页面设计

“学习计划”页新增教材任务树：

- 只显示已勾选科目的树。
- 章节点显示标题、完成数量、总数量和百分比。
- 节节点显示完成数量、总数量和当前状态。
- 叶子节点显示编号、标题、预计分钟数、重要程度、来源链接和掌握按钮。
- 叶子节点可直接打卡 `已学`、`模糊`、`掌握`。
- `模糊` 节点进入复盘队列，并在冲刺期优先回炉。

“今日学习”页仍显示少量今日推荐任务，避免被整棵树淹没。今日任务标题必须精确到教材目录节点，例如：

`管理 / 第一章 工程造价管理及其基本制度 / 第一节 工程造价基本内容 / 工程造价及计价特征`

## 数据结构

教材目录树数据需要是稳定的本地常量，字段包括：

- `subjectId`
- `sourceUrl`
- `chapters[]`
- `sections[]`
- `items[]`
- `id`
- `chapterTitle`
- `sectionTitle`
- `title`
- `estimatedMinutes`
- `importance`

兼容旧数据：已有 `knowledgeStatus` 继续使用叶子节点 `id` 作为 key。旧版本数据加载时自动补齐新增节点，已有完成状态保留。

## 验收标准

- 学习计划页能以树形结构显示四科教材目录任务。
- 章、节、叶子节点都能清楚显示完成/未完成状态。
- 今日任务来自教材目录树的下一个未完成或模糊叶子节点。
- 打卡叶子节点后，树状态、今日任务、角色经验、科目进度同步更新。
- 2026-09-17 以后不再生成新教材节点任务。
- 旧数据可正常打开，并自动补齐新增教材节点状态。
