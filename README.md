# 造价勇者：一级造价师 RPG 学习计划

本项目是一个 Windows 本地运行的一级造价师学习推进网页。它面向 2026 年一级造价工程师（土木建筑工程）备考，核心能力包括官方教材目录树任务、科目勾选、角色等级 EXP、外部题库入口、资料链接、RPG 复盘和外部考试得分记录。

## Windows 使用

1. 安装 Node.js LTS: <https://nodejs.org/>
2. 解压项目压缩包。
3. 双击 `start-yizaostudy.cmd`。如果想用中文入口，也可以双击 `启动学习计划.bat`。
4. 浏览器会打开 `http://127.0.0.1:4173`。

学习数据保存在解压目录的 `data/app-data.json`。备份时复制整个项目目录，或至少复制 `data/app-data.json`。

如果启动窗口闪退或没有打开网页，先重新双击 `start-yizaostudy.cmd`。新版启动脚本会把错误留在窗口中，并在同目录生成 `startup.log`，常见原因是没有安装 Node.js、Node.js 版本低于 20，或 4173 端口已被其他程序占用。

## 学习计划规则

- 今日任务来自教材目录树的下一个未完成节点，例如“管理 / 工程造价管理及其基本制度 / 工程造价基本内容 / 工程造价及计价特征”。
- `学习计划` 页按 `科目 -> 章 -> 节 -> 目/知识点` 展示树形任务列表，并显示完成数量和百分比。
- 今日学习任务可以标记为 `已学`、`模糊`、`掌握`，也可以点 `推迟`。`推迟` 只推迟当天任务，不改变知识点状态，不加 EXP，也不会让教材知识点遗漏。
- 教材树里的 `撤销标记` 用于处理误点，会把知识点回到 `未开始`；它不是取消学习任务。
- `已学` 表示接触过内容；`模糊` 和 `掌握` 会先弹出自检提示，让学习者做一次主动回忆后自行确认。标记为 `模糊` 的节点会进入复盘队列。
- 复盘页会给出 `Boss 雷达`、`弱点副本`、`回忆挑战`、`案例联动提醒` 和 `下一步动作`，避免只显示空泛统计。
- `2026-09-17` 起进入最后一个月冲刺期，不再推进新知识点，主要安排真题、套卷、错题和薄弱点回炉。
- 案例分析不是最后一个月才开始：系统会按管理、计价、计量的前置完成情况提示案例影子任务，先做思路拆解，后期再做限时整题。

教材目录根据公开教材目录和考试大纲资料整理，只保存目录级任务和来源链接，不复制教材正文。

## 资料与题库

- `资料库` 内置官方大纲、中国人事考试网、浙江人事考试网、教材目录、公开视频/资料和真题资料下载入口。
- `题目练习` 只在存在公开题库资源时显示，目前提供 233 网校、环球网校、万题库等外部入口；系统不抓取、不复制题目正文。
- 用户可以手动记录外部练习摘要、错因、来源链接和模考得分。模考得分会增加 EXP。

## 设计边界

- 本地单用户，电脑浏览器优先。
- 不直连粉笔账号，不批量搬运付费或受限题库内容。
- 公开题源以来源链接和用户手动导入为主。
- 后续可扩展公网访问、登录、HTTPS、自动备份和半自动题目解析。
